Kiani Life & Health recruitment site — dark theme with gold accents for insurance agent recruiting.

## Design System
- Fonts: Playfair Display (serif/display), Outfit (sans/body)
- Colors: Dark ink bg (#07090F), gold accent (#B8963E), gold-hi (#D4AF6A)
- Tokens: --ink, --deep, --panel, --gold, --t1, --t2, --t3
- No border-radius (sharp edges throughout)

## Purpose
- B2B recruitment site targeting young people to become insurance sellers
- Lead magnet: "Book a Discovery Call with Moiz Kiani" as primary CTA
- Showcases income potential, training, flexibility, career growth
- NOT a consumer-facing insurance site

## Key Decisions
- All four selling points: commissions, training, flexibility, career growth
- Income proof: Both numbers + success stories
- Primary CTA: Book a call (not apply form)
- Commissions page: password-protected (auth to be added later via Cloud)
- Agent Dashboard: future feature requiring Cloud (login + daily activity tracking)

## Pages
- / (home): Hero, WhyKiani, Earnings, Process, Testimonials, MeetTheTeam, Partners, Apply, FAQ, CTA
- /commissions: Protected commission grid (placeholder lock for now)
- /training: Training system + weekly proof breakdowns
- /career-growth: Agent → Agency Owner career path
- /leads: Warm lead system details
- /bonuses: Fast-track bonuses & incentive trips
- /partners: Ecosystem partners (RevUp, Orca BPO, KLH)

## FAQ Removals
- REMOVED "generate your own referral pipeline" (scares candidates re: friends/family selling)
- REMOVED "field work" (implies door-to-door)
- REMOVED specific income promises like "$1,000 in first month"
- REPLACED with "Many new agents begin earning attractive paychecks within their first month"
